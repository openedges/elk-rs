ELK-RS Demonstrators

Automatic layout algorithms compute layouts for diagrams. Powered by elk-rs — a pure Rust port of the Eclipse Layout Kernel, running in your browser via WebAssembly.

Interactive Editor

Interactively edit a textual description of a graph while a corresponding diagram is generated on-the-fly. Choose between two formats:

elkt

ELK's textual graph DSL. Layout is performed in-browser using elk-rs WASM.

json

ELK's JSON format. Layout is performed in-browser using elk-rs WASM.

Examples

An annotated collection of example graphs illustrating various configuration possibilities.

Source: ELK's models repository.

Technologies

elk-rs
Pure Rust port of ELK, compiled to WebAssembly for in-browser layout.
Monaco Editor
The code editor that powers VS Code.
Eclipse Layout Kernel
The original Java-based layout algorithms that elk-rs reimplements.
